# danny-tools

A collection of various productivity tools, scripts, and macOS utilities created by **Danny Do**.

---

## 🛠️ Tools & Projects

| Project | Type | Description |
|-----------|------|-------------|
| [**MenuBarDotSeparator**](./MenuBarDotSeparator) | macOS App | A lightweight menu bar utility that lets you organize icons using a dot (`•`) separator. |
| [**ag-quota**](./extensions/ag-quota) | VS Code Extension | Shows live AI model quota (Flash, Pro, Claude) for the Antigravity IDE directly in the status bar. |

---

## Contributing

Pull requests welcome! To add a new tool or project, simply create a new folder at the root level (or under `extensions/` if applicable) and include its own `README.md`.
